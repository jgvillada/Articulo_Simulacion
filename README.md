# Modelos de Simulación – Análisis y mejora del sistema de atención al cliente

## Descripción

Este repositorio contiene los modelos de simulación desarrollados en **SIMUL8** para el artículo:

> **"Análisis y mejora del sistema de atención al cliente en un punto de venta de alimentos mediante simulación de eventos discretos".**

El trabajo fue desarrollado inicialmente en el curso **Simulación (2025-1)** y posteriormente refinado para su presentación a la **Revista UIS**.

## Contenido del repositorio

Los archivos corresponden a tres modelos de simulación y sus respectivas versiones de calibración y validación.

### Modelo 1 – Sistema base

Representa el funcionamiento actual del sistema estudiado y constituye el escenario de referencia para el análisis.

**Archivos:**
- `MODELO1.S8`
- `MODELO1.bk8`
- `Calibracion_Modelo1.S8`
- `Calibracion_Modelo1.bk8`
- `Validacion_Modelo1.S8`
- `Validacion_Modelo1.bk8`

---

### Modelo 2 – Propuesta de mejora 1

Corresponde a la primera alternativa de mejora planteada para aprovechar el desempeño del sistema.

**Archivos:**
- `MODELO2.S8`
- `MODELO2.bk8`

---

### Modelo 3 – Propuesta de mejora 2

Representa una segunda alternativa de mejora, incorporando modificaciones adicionales respecto al Modelo 2. De acuerdo con los resultados obtenidos, este escenario presenta un desempeño superior al de la propuesta anterior.

**Archivos:**
- `MODELO3.S8`
- `MODELO3.bk8`
- `Calibracion_Modelo3.S8`
- `Calibracion_Modelo3.bk8`
- `Validacion_Modelo3.S8`
- `Validacion_Modelo3.bk8`

## Estructura de archivos

- **`.S8`**: modelos de simulación de SIMUL8.
- **`.bk8`**: archivos de respaldo generados automáticamente por SIMUL8 al ejecutar el modelo.

## Objetivo

Este repositorio tiene como propósito almacenar y documentar los modelos empleados durante el proceso de construcción, calibración, validación y evaluación de escenarios de mejora del sistema analizado, facilitando la reproducibilidad y el seguimiento del estudio.
